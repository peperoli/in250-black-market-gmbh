# Folder Structure

## Top-level Folders

| Folder  |                          Description                          |
| :-----: | :-----------------------------------------------------------: |
|   src   |                Contains source code files[^1]                 |
|  docs   | Documentation files (currently in DOC, only MD in the future) |
| images  |                 Image files (JPEG, PNG etc.)                  |
| website |         Contains our website, use for HTML, CSS etc.          |
| secrets |                      Super secret stuff.                      |

[^1]: Currently every version of cowsay-python has it's own folder (this should probably change).
